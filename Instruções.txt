() = o que você tem que escrever
[] = o que o comando faz

git clone (endereço) [baixa o repositório do endereço]
git status [Exibe status do projeto]

*[modificar os arquivos]

git add (nome dos arquivos que você mudou)
ou
git add -A [manda tudo]

git commit -m "(Comentário sobre a modificação)" [cria o pacote de modificações]

git push origin master [manda as modificações]
